# OxoBrowser
  
已更换为cef的嵌入的chrome核心，目前支持舰娘二期的显示。
提供数据接口方便其他游戏改造接入。 
已实现部分游戏提取、顶置、代理、关闭声音、缩放比例等基本功能。

### 如何接入数据部分
HttpPacketHookManager中可以设置需要过滤的网站地址及请求类型，以及处理函数
